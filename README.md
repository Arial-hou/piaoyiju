# 漂居 - 城市漂泊青年的决策伙伴

一个帮助城市租房青年做消费决策的AI工具。

## 功能特性

- 🏠 **漂居指数**：综合评估稳定度、归属感、经济感
- 🛒 **好物推荐**：根据租期、预算、习惯智能推荐
- 🔄 **买or租决策**：大件物品购买与租赁成本对比
- 📦 **搬家转卖**：一键生成闲鱼转让文案

## 技术栈

- React 18 (CDN)
- Tailwind CSS
- 纯前端实现，无需后端

## 使用方式

直接在浏览器中打开 `index.html` 文件即可使用。
